# スケジュールプランナー

[English](README.md)

時間帯が少し変動する定期的なスケジュールを計画するためのツール。例えば、毎日16:00に開始、16:10に本編、16:50-17:00に終了を予定している場合、開始が遅れたり早まったりしても自動的に時間を調整できます。

# 使い方
ドロップダウンからテンプレートを選択するか、テキストボックスに独自のものを入力できます。

## テンプレート形式
テンプレートは以下のプロパティを持つJSONオブジェクトです：

- `default-start`: "HH:MM"形式（24時間表記）のデフォルト開始時間を表す文字列
- `times`: 各アクティビティが以下のプロパティを持つアクティビティオブジェクトの配列：
  - `default-length`: 分単位のデフォルト継続時間を表す数値
  - `description`: アクティビティを説明する文字列

### 例
```json
{
  "default-start": "16:10",
  "times": [
    {
      "default-length": 5,
      "description": "Opening ceremony"
    },
    {
      "default-length": 10,
      "description": "Main discussion"
    }
  ]
}
```

開始時間と終了時間を選択し、Generateをクリックしてください。
